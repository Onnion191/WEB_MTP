1.6 EXEMPLE: DISSENY I IMPLEMENTACIÓ CLASSE FRACCIÓ

Aquest exemple consistirà en el disseny i implementació d'una calculadora de fraccions, que operarà amb la classe Fracció. El programa farà el següent:
  • Es llegeixen dues fraccions.
  • S'escull l'operació a realitzar
    • +: suma
    • -: resta
    • *: multiplicació
    • /: divisió
  • Les fraccions sempre es presenten en forma simplificada i denominador estrictament positiu

Fraccio.h

Nom de la classe:

Fraccio

Descripció de la classe:

denominador>0 i simplificada.
  • ho compliran tots els objectes de la classe
  • no cal tornar-ho a indicar a les precondicions dels mètodes públics, ho han de complir tots els objectes de la classe

Interfície:

inclou constructors, suma, resta, multiplicació, divisió i mostrar

Mètodes privats:

  • simplificar (mètode d'instància)
  • calcular MCD (mètode de classe --> static)
  • calcular MCM (mètode de classe --> static)

Mètodes de classe

Els mètodes qualificats com a static són mètodes de classe:
  • corresponen a les accions i funcions ja conegudes
  • no s’executen sobre cap objecte i s’utilitzen de manera restrictiva
  • es poden cridar des de
    • dins de la classe nom_mètode_classe(paràmetres);
    • fora de la classe (caldria que fos públic) Nom_classe::nom_mètode_classe(paràmetres);
El mètode que calcula el MCD és de classe. És una funció que es pot usar per implementar mètodes de la classe Fraccio.

Fraccio.h resultant

class Fraccio {
// Descripció: una fracció amb denominador>0 i simplificada
public:
    // Constructors
    Fraccio();             // Pre: --;   Post: ha creat fracció 0/1
    Fraccio(int n, int d); // Pre: d!=0; Post: ha creat fracció en format (-)n/d i simplificada
    // Consultors
    Fraccio suma(Fraccio f) const;
    // Pre:--; Post: retorna la suma simplificada entre fracció actual i f
    Fraccio resta(Fraccio f) const;
    // Pre:--; Post: retorna la resta simplificada entre fracció actual i f
    Fraccio producte(Fraccio f) const;
    // Pre:--; Post: retorna el producte simplificat entre fracció actual i f
    Fraccio divisio(Fraccio f) const;
    // Pre: f no val zero; Post: retorna la divisió simplificada entre fracció actual i f
    void mostrar() const;
    // Pre:--; Post: mostra la fracció actual en format (-)num/den o (-)num si denominador=1
private:
    int a_num; // conté el signe de la fracció
    int a_den; // serà major que zero
    void simplificar();
    // Pre: --; Post: ha simplificat la fracció actual
    static int maxim_comu_divisor(int a, int b);
    // Pre: a,b > 0; Post: retorna M.C.D.(a,b)
    static int minim_comu_multiple(int a, int b);
    // Pre: a,b > 0; Post: retorna M.C.M.(a,b)
};
