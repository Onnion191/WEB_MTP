Parametric constructor of the Window class. All parameters are optional, so if you choose to instantiate the object without parameters, default values will be used instead.

Parameters
widthPixel width of the screen. If empty, DEFAULT_WIDTH is assumed.
heightPixel height of the screen. If empty, DEFAULT_HEIGHT is assumed.
background_colorRGBColor that will be used as background color. Its value is BLACK by default.
drawing_colorRGBcolor that will be used to draw the primitives and Turtle interaction with the screen. WHITE is used by default.
line_widthWhen drawing some figures, the width of the line can be chosen. Width 1 is used by default.
in_radiansBy default, Window uses degrees to represent angles. If this parameter is true, the Window will represent angles in radians.
fWhen building the Window object, you can choose the default font that will be used when printing text. You can add more fonts later, without having to delete the others. If empty, DEFAULT_FONT is used instead.
log_sizeThe size of the log can be adjusted by the user. By default, it's DEFAULT_LOG_SIZE.
