bool Window::addTurtle ( std::string  turtle_name)

Adds a Turtle to the Window. With this function, the name of the turtle is the only parameter that can be chosen. The pen (PaintBrush) will be the same as the Window one, the start point will be TURTLE_HOME and the Font will be the active Font of the Window.

Parameters
turtle_nameName of the Turtle.
Returns
A boolean value, true if the Turtle has been successfully added, false otherwise.
