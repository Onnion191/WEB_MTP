◆ text() [1/6]

void Window::text ( Font &  font,
float  x,
float  y,
std::string  text,
int  flags = 0 
)

Puts a text into the screen. By default, the text is aligned to the left. Depending on the alignment, the point passed will represent different positions of the text box. When aligned left, the point passed will be the top left corner of the text box. When aligned right, it will be the top right corner, and when aligned to the center, the point will represent the exact center of the text box. For example this code:

#include "WindowAddon.h"
int main(int argc, char **argv){
Window w(201, 201); //White background color and black drawing color by default
Font f("OpenSans-Regular.ttf", BLACK, 12, 0);
w.text(f, 101, 101, "Your text goes here", TEXT_ALIGN_CENTER);
w.waitWindowCloseButton();
return 0;
}

Would generate this output:

Parameters
fontThe font used to write the text.
xX coordinate where the text will be drawn
yY coordinate where the text will be drawn.
textThe text that will be written.
flagsThis parameter can be used to specify the alignment of the text. See Text Alignment Options for more details. Its value is 0 by default.

#include "WindowAddon.h"
