Detailed Description

Mid-level interface class that allows you to access all the functionalities of the library.

This is the main class of the library. It contains all the drawing methods, and encapsulates all the needed modules to operate the display and its features like the Turtles. Look for primitives or turtle drawing in documentation for more detail about drawing functionalities. It also allows you to store some Fonts and access them by the name you choose.
