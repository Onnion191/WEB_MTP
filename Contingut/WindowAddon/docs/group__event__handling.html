◆ waitAnyKeypress()

int Window::waitAnyKeypress ( bool  on_release = false)

Waits for the next WindowEvent with the WindowEventType WINDOW_EVENT_KEY_UP or WINDOW_EVENT_KEY_DOWN, depending on on_release parameter. In other words, it waits until any key is pressed.

Parameters
on_releaseWhen true, the function waits for a "release" event, so it will wait for WINDOW_EVENT_KEY_UP, otherwise will wait for WINDOW_EVENT_KEY_DOWN. By default its value is false.
Returns
The keycode of the pressed key.

Waits for the next WindowEvent with the WindowEventType WINDOW_EVENT_KEY_UP or WINDOW_EVENT_KEY_DOWN, depending on on_release parameter and only waits for the selected key, ignoring the other keypresses. In other words, it waits until the selected key is pressed.

Waits for the next WindowEvent with the WindowEventType WINDOW_EVENT_MOUSE_BUTTON_UP or WINDOW_EVENT_MOUSE_BUTTON_DOWN on the selected mouse button by the user, depending on on_release parameter. In other words, it waits until the selected mouse button is pressed.

Parameters

on_releaseWhen true, the function waits for a "release" event, so it will wait for WINDOW_EVENT_MOUSE_BUTTON_UP, otherwise will wait for WINDOW_EVENT_MOUSE_BUTTON_DOWN. By default its value is false.

WindowEvent Window::waitWindowEvent ( )

Waits for the next WindowEvent, without filters. Be careful, this method waits forever. If no event is triggered, the program will go on forever, without exiting this method. It's more recommendable to use waitWindowEventTimed(float) to not to wait forever.

Returns
The first WindowEvent in the EventQueue of the Window.

◆ waitWindowEvent() [2/2]

WindowEvent Window::waitWindowEvent ( WindowEventType  ev_type)

Waits for the next WindowEvent with the type specified by the user. Be careful, this method waits forever. If no event is triggered, the program will go on forever, without exiting this method. It's more recommendable to use waitWindowEventTimed(WindowEventType, float) to not to wait forever. The events with different type are skipped and removed from the queue.

Parameters
ev_typeType of the target event.
Returns
The first WindowEvent in the EventQueue of the Window with the same type as ev_type.
