Windows Downloads

Check the How to install the Window Add-on to learn how to use these files. The version numbers inside () tells you what version of the prerequisites is needed to get the downloaded files working.

  • Window Add-on headers (used when compiling Window Add-on from source code)
  • MinGW pre-compiled Allegro binaries (5.2.4) (GCC 5.1.0)
  • MinGW pre-compiled Window Add-on binaries (1.1) (GCC 5.1.0/Allegro 5.2.4)
  • MinGW pre-compiled Window Add-on binaries (1.1) + pre-compiled Allegro 5.2.4 binaries (GCC 5.1.0). With this download you simply have to drag and drop all the files inside your MinGW installation and you'll have both Allegro and Window Add-on properly installed.

Linux Downloads

You'll find all Linux installation files inside installation/Linux folder in our GitHub repository. Use

git clone https://github.com/EnricRG/PID-UdG

and follow the instructions in the readme.txt on that folder, or follow the Linux Installation Guide.
