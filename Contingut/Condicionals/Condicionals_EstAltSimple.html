Estructura Alternativa Simple If - else

Una estructura alternativa és aquella que permet tenir diferents execucions en funció de l'estat del programa. Per determinar quina execució cal dur a terme per un estat determinat del programa, utiltizem condicions. Tota condició té assignada un tall de codi, de manera que quan la condició és certa, aleshores s'executa el tall de codi. I quan la condició és falsa, simplement s'ignora aquest tall.
Els llenguatges de programació proporcionen dos tipus de condicions possibles:
  • if(condició): Aquest cas ens permet entrar una condició, la que nosaltres decidim. Per exemple: i == 4. Aquesta estructura demana que la condició retorni un valor booleà, és a dir, que només podem utlitzar operadors booleans per efectuar la condició. Si la condició falla, aleshores salta tot el que estigui compres dins de l'estructura, i executa el else, si n'hi ha.
  • else: Aquesta estructura s'executa quan la condició del if falla. Així doncs, sempre ha d'anar a posteriori d'una condició if.
Per poder visualtizar millor el seu funcionament, mirem el següent exemple: Es fa una "loteria" on es demana al usuari que encerti el nombre amagat. 

        //Entrada: un natural entre 1 i 10. 
        //Sortida: mostra si ha encertat el nombre secret o no.
            unsigned int nombre_amagat = 3;

            cout<<"Entra un natural entre 1 i 10"<<endl;
            unsigned int a;
            cin>>a; 

            //Presa de decisio:

            if(a == nombre_amagat)//Si el nombre que ha entrat és igual a nombre_amagat = (3) |(condició a == 3)|
                cout<<"Has encertat el nombre secret"<<endl; 
            else //Si no es igual a 3
                cout<<"Ho sento, no has encertat el nombre secret"<<endl;

Amb l'exemple anterior com a base, intentarem fer l'exemple de l'ascensor (veure pàgina 76 del llibre).Hem d'indicar quin llibre és.
L'exercici ens diu el següent:
Volem fer el programa que controla un bloc de pisos que li falta el pis numero 13. Aleshores, cal compensar aquest pis que manca per tal que l'ascensor arribi correctament a tots els pisos. Tenim dues formes de resoldre el problema:
  • Amb una estructura simple completa (if - else): utilitzant l'esquema anterior podem resoldre l'exercici. 
    
               algorisme Control_ascensor 
                   natural pis
                   llegir pis
                   natural pis_real 
                   si pis>13
                      pis_real=pis-1
                   altrament
                      pis_real=pis
                   escriure pis_real
            
    
               // Entrada: llegeix un nombre natural (pis) diferent de 13
                // Sortida: escriu el pis real (pis_real) al qual ha d'anar l'ascensor en funció del pis entrat

                #include <iostream>
                using namespace std; 

                int main() {
                   unsigned pis;
                   cout<<"Pis: ";
                   cin>>pis;
                   unsigned pis_real;
                   if (pis>13) {
                      pis_real=pis-1;
                   }
                   else {
                      pis_real=pis;
                   }
                   cout<<"L'ascensor pujarà al pis real "<<pis_real<<endl;
                   return 0;
                }
  • Amb una estructura simple incomplerta ( if): aquesta estructura li falta l'else o altrament. Per això es diu que és incomplerta.
    
              algorisme Control_ascensor 
                   natural pis
                   llegir pis
                   natural pis_real
                   pis_real=pis
                   si pis>13
                      pis_real=pis-1
                   escriure pis_real
            
    
               // Entrada: llegeix un nombre natural (pis) diferent de 13
                // Sortida: escriu el pis real (pis_real) al qual ha d'anar l'ascensor en funció del pis entrat

                #include <iostream>
                using namespace std; 

                int main() {
                   unsigned pis;
                   cout<<"Pis: ";
                   cin>>pis;
                   unsigned pis_real;
                   pis_real=pis;
                   
                   if (pis>13)
                      pis_real=pis-1;
                   
                   cout<<"L'ascensor pujarà al pis real "<<pis_real<<endl;
                   return 0;
                }

Us de variables booleanes

Fins el moment hem vist que les condicions són expressions boleanes que si es compleixen s'entra dins del condicional. El que està passant realment és que el resultat d'evaluar l'expressió és un booleà, i aquest valor s'utilitza per informar al condicional de si cal entrar o no.
Així doncs, el que realment necessitem és un valor booleà, ja sigui fruit d'una expresió o bé una variable booleana.
Un exemple seria el següent: 

            // Entrada: llegeix un nombre natural (any) 
            // Sortida: escriu si any és o no un any de traspàs 

        /************************************************************************
                                EXPRESSIÓ LÒGICA
        /************************************************************************
            algorisme Any_traspàs 
               natural any
               llegir any 
               si (any mod 400==0) o (any mod 4==0 i any mod 100!=0)
                  escriure any, " és de traspàs" 
               altrament 
                  escriure any, " no és de traspàs"


        /************************************************************************
                                VARIABLE BOOLEANA 
        /************************************************************************
        algorisme Any_traspàs 
           natural any
           llegir any
           booleà traspàs
           traspàs=(any mod 400==0) o (any mod 4==0 i any mod 100!=0)
           si traspàs
              escriure any, " és de traspàs"
           altrament
              escriure any, " no és de traspàs"
Per veure l'exemple en C++: 

                // Entrada: llegeix un nombre natural (any) 
                // Sortida: escriu si any és o no un any de traspàs 
                
                #include <iostream>
                using namespace std;                

                int main() {
                    unsigned any;
                    cout<<"Any gregorià: ";
                    cin>>any;
                    bool traspas= (any % 400==0) or (any % 4==0 and any % 100!=0);
                    if (traspas)
                        cout<<"L'any "<<any<<" és de traspàs"<<endl;
                    else
                        cout<<"L'any "<<any<<" no és de traspàs"<<endl;
                    return 0;
                }

Exemple2: Màxim i mínim de dos enters

Es demana que l'usuari entri dos valos enters, i el programa digui quin és el màxim i quin el mínim 

       
        // Entrada: llegeix dos nombres enters (nombre1,nombre2)
        // Sortida: escriu màxim(nombre1,nombre2) i mínim(nombre1,nombre2)

        algorisme Màxim_mínim 
           enter nombre1, nombre2
           llegir nombre1
           llegir nombre2
           enter max, min
           si nombre1>=nombre2
              max=nombre1 
              min=nombre2
           altrament
              max=nombre2 
              min=nombre1
           escriure max,min 

        

            // Entrada: llegeix dos nombres enters (nombre1,nombre2)
            // Sortida: escriu màxim(nombre1,nombre2) i mínim(nombre1,nombre2)

            algorisme Màxim_mínim 
               enter nombre1, nombre2
               llegir nombre1
               llegir nombre2
               enter max, min
               max=nombre1 
               min=nombre2
               si nombre1<nombre2
                  max=nombre2 
                  min=nombre1
               escriure max,min 

        

            // Entrada: llegeix dos nombres enters (nombre1,nombre2)
            // Sortida: escriu màxim(nombre1,nombre2) i mínim(nombre1,nombre2)
            
                #include <iostream>
                using namespace std; 

                int main() {
                    int nombre1, nombre2;
                    cout<<"Primer nombre enter: ";
                    cin>>nombre1;
                    cout<<"Segon nombre enter: ";
                    cin>>nombre2;
                    int max, min;
                    if (nombre1>=nombre2) {
                        max=nombre1;
                        min=nombre2;
                    }
                    else {
                        max=nombre2;
                        min=nombre1;
                    }
                    cout<<"Màxim de "<<nombre1<<" i "<<nombre2<<" és "<<max<<endl;
                    cout<<"Mínim de "<<nombre1<<" i "<<nombre2<<" és "<<min<<endl;

                    return 0;

                }

            

            // Entrada: llegeix dos nombres enters (nombre1,nombre2)
