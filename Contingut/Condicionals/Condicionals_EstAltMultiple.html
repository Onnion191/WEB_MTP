3.3 ESTRUCTURA ALTERNATIVA MÚLTIPLE

Estructura alternativa múltiple

Per començar amb l'estructura Alternativa múltiple, cal haver entès els continguts de l'estructura alternativa simple.
Fins ara hem vist com avaluar una expressió i determinar una execució o una altre en funció de si es compleix la condició establerta. Però a vegades es poden donar diferents casos que necessitem avaluar diferents solucions. En aquests casos apareix l'estructura alternativa múltiple.

Declaració i funcionament

Una forma de fer una estructura alternativa múltiple és encadenar diferents estructures simples fent així una cascada de condicionals que pot arribar fins a tants nivells com es desitji. D'aquesta pràctica se'n diuen condicionals niats.
Tot i això, també apareix el altrament si, o else if que substitueix els condicionals niats per una altre nomenclatura i que simplement serveix com a "sintantic sugar". És a dir, a nivell d'execució no hi ha diferència entre ells, però si que pot ajudar a fer més net el codi.
A continuació veurem un exemple de cada:

Condicionals niats

si(condició 1)
        si (condició 1.1)
            fer quelcom
        altrament
            fer quelcom
altrament
    si(condició 2)
        fer quelcom
    altrament
        si(condició 3)
            si (condició 3.1)
                fer quelcom
            altrament

Altrement si:

si(condició1)
    fer quelcom
altrament si(condició 2)
    fer quelcom 2
altrament si (condició 3)
    fer quelcom 3

    fer quelcom n

Switch

Una altre estructura que podem utilitzar és el switch. Aquesta estructura ens permet fer diferents execucions en funció del valor d'una variable.
L'estructura té la següent forma: 
unsigned int digit;
switch(digit){
    case 0: cout<<"Numero 0"; break;
    case 1: cout<<"Numero 1"; break; 
    case 2: cout<<"Numero 2"; break;
    case 3: cout<<"Numero 3"; break;
    default: cout<<"Numero major que 3"; break;                
        }
El seu funcionament és el següent: Agafa el valor de la variable "digit" i comprova un a un cada " case " fins a trobar el valor que li correspon.
És a dir, primer comprova si digit == 0, després digit == 1, i així successivament fins a trobar la condició correcta, aleshores executa el case corresponent, i tots els posteriors.
Si no troba la condició correcta, aleshores executarà la branca "default".
Al final de cada case, cal utilitzar la comanda "break", ja que de no ser així, l'estructura seguiria executant tots els casos posteriors a ell.

Exemple 1:

Es demana un programa que digui quins efectes tindrà un terratrèmol segons l'escala de Richtard. Cal que l'usuari entri el nivell del terratrèmol que desitja, i el programa retorni els efectes que tindrà. A continuació hi ha la informació de cada nivell:
Per veure la solució: 
//Entrada: l'usuari entra un nombre real entre  0-10
//Sortida: mostra els efectes del terratrèmol.

#include <iostream>
using namespace std; 

int main(){             

    double richter; 
    cout<<"Entra el nivell del terratremol,(0-10)"<<endl;
    cin>>richter;
    
    if (richter>=8.0)
        cout<<"Most structures fall";
    else if (richter>=7.0)
        cout<<"Many buildings destroyed";
    else if (richter>=6.0)
        cout<<"Many buildings considerably damaged, some collapse";
    else if (richter>=4.5)
        cout<<"Damage to poorly constructed buildings";
    else
        cout<<"No destruction of buildings";
    
    //Què passaria si es canviés l'ordre de les condicions?
    
    return 0;
                    }
