EXEMPLES D'OPERANDS BOOLEANS


Volem fer un programa que ens digui si hem aprovat una assignatura o no. Per fer-ho tenim en compte dues coses: primer de tot cal que la mitjana de l'assignatura sigui igual o més gran que 5. Segona, cal haver superat la nota mínima de l'examen, que normalment és un 4.
Cal que el l'usuari entri la seva mitjana i la seva nota al examen 

        #include <iostream>
        using namespace std;
        
        int main(){
            unsigned int minExamen = 4;//La nota mínima al examen és un 4;
            
            unsigned int nota = 0; 
            cout<<"Entrala teva nota ";//Demanem al usuari que entri la seva nota
            cin>>nota; ///ALERTA, QUE PASSA SI L'USUARI ENTRA UNA NOTA NEGATIVA? 
            
            unsigned int mitjana = 0;
            cout<<"Entrala teva mitjana ";//Demanem al usuari que entri la seva mitjana
            cin>>mitjana; ///ALERTA, QUE PASSA SI L'USUARI ENTRA UNA MITJANA NEGATIVA? 
            
            if (nota > 10 || mitjana > 10){//Si la nota o la mitjana és superior a 10 es que l'usuari ens vol enganyar
                cout<<"La teva nota o mitjana no es valida"<<endl;
                return 0; //Acabem la execucio
            }
            else//Si la nota i la mitjana  es mes petita que 0
            {}//No cal fer res, podríem suprimir aquest else. (també podríem fer tota la resta del codi dins del else, però no és recomenable)
            
            //Comprovem que té l'examen aprovat
            if(nota >= minExamen && mitjana >=5)//Si té l'examen aprovat i la mitjana aprovada
            {
                cout<<"Felicitats has aprovat"<<endl;
            }
            else 
            {
                cout<<"La mitjana o l'examen no arriben a la nota minima. "<<endl;
            }
            return 0;

        }


Estructures Encadenades


Agafant l'exercici anterior, ens demanen que informem a l'alumne que ha suspès, per quin motiu ha suspès. Per fer-ho, cal que comprovem una a una les condicions de suspèns i identificar quina d'elles és la que falla. 

        #include <iostream>
        using namespace std;
        
        int main(){
            unsigned int minExamen = 4;//La nota minima al examen es un 4;
            
            unsigned int nota = 0; 
            cout<<"Entrala teva nota ";//Demanem al usuari que entri la seva nota
            cin>>nota; ///ALERTA, QUE PASSA SI L'USUARI ENTRA UNA NOTA NEGATIVA? 
            
            unsigned int mitjana = 0;
            cout<<"Entrala teva mitjana ";//Demanem al usuari que entri la seva mitjana
            cin>>mitjana; ///ALERTA, QUE PASSA SI L'USUARI ENTRA UNA NOTA NEGATIVA? 
            
            if (nota > 10){//Si la nota és superior a 10 es que l'usuari ens vol enganyar
                cout<<"La teva nota no és valida"<<endl;
                return 0; //Acabem la execucio
            }
            else//Si la nota es mes petita que 0
            {
                if(mitjana > 10)
                {
                    cout<<"La teva mitjana no és valida"<<endl;
                    return 0; //Acabem la execucio
                }
                else 
                {}//No cal fer res
            }
            
            //Comprovem que te l'examen aprovat
            if(nota >= minExamen)//Si te l'examen aprovat
            {
                if(mitjana >= 5)//Si te la mitjana aprovada
                {
                    cout<<"Felicitats has aprovat"<<endl;
                }
                else//Si te la mitjana suspesa
                {
                    cout<<"La mitjana no t'arriba al 5"<<endl;
                }
            }
            else //Te l'examen suspes
            {
                cout<<"l'examen no arriba a la nota minima. "<<endl;
            }
            return 0;

        }
