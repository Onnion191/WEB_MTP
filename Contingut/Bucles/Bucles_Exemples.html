Programa 5

Ens demanen sumar els nombres naturals de 1 fins a n. On n és un valor entrat per l'usuari. Cal tenir present les següents consideracions:
  • Hem de fer un bucle amb "n" iteracions. (Nombre fixat d'iteracions).
  • Cal que l'usuari entri el nombre "n", el qual ha de ser natural.
  • Cal tenir una variable que vagi guardant el valor del sumatori en cada iteració.

    #include <iostream> 
    using namespace std;
    int main(){
        //Entrada de dades: 
        unsigned int n; 
        cout<<"Entra un nombre natural: ";
        cin>>n; 
        
        
        //Validació
        if(n < 0){//l'usuari no ha entrat un nombre natural. 
            cout<<No has entrat un nombre natural<<endl;
            return 0;//Acabem la execució
        }
        //Càlcul
        unsigned int suma = 0; 
        for(unsigned int i = 1; i <=n ; i++)
            suma+=i; 
    
        //Sortida
        cout<<"El valor de la suma es: "<<suma<<endl;
        
        //Finalització
        return 0;
    }
