Qüestió 2

Considera dos subprogrames les capçaleres dels quals són: 
void mostra_text(string text);
int pow(int base, int exponent);
D’acord amb aquestes capçaleres digues si són correctes o no les següents crides: 
  1. cout<<mostra_text("Hola")<<endl;
  2. cout<<pow(2,3)<<endl;
  3. if(mostra_text("Adéu") < mostra_text("Hola"))
      cout<<"Adéu"<<endl;
  4. if("Adéu" < "Hola")
      mostra_text("Adéu");
  5. cout<<"Entra dos enters"<<endl;
int a, b;
cin>>a>>b;
if(pow(2,3)<pow(a,b))
    mostra_text("El de l’usuari és major");


  1. No, un coutno pot portar enmig una crida a una acció; si retornés un stringsí que es podria fer, però així no.
  2. Sí, el coutmostra l’intretornat per la funció pow.
  3. No, mostra_textés una acció; no retorna res i per tant no es pot comparar amb res.
  4. Sí, es comparen els stringsentre ells i es mostra el text corresponent amb l’acció mostra_text.
  5. Sí, es comparen els enters i es mostra el text corresponent.

int total_centims = unsigned(100*total+0.5);
int total_centims_iva= unsigned(100*total*IVA+0.5);


En les dues instruccions s’arrodoneix una quantitat a cèntims, per tant fem un acció (a_centims) que ho faci. L’acció ha de rebre una quantitat real d’entrada (pas per valor) i ha de retornar un nombre natural. 
unsigned a_centims(doublequantitat){

	return 100*quantitat+0,5;

int total_centims=a_centims(total);
int total_cèntims_iva= a_cèntims(total*IVA);

Qüestió 5

5. A continuació hi ha tres accions que pretenen intercanviar el valor de dues variables. Analitza si aconsegueixen el seu propòsit. 
  1. void swap1(int& a, int& b){	
     a=b;
     b=a;
}
  2. void swap2(int a, int b){	
      int aux=a;	
      a=b;
      b=a;
}
  3. void swap3(int& a, int& b){	
       a=a+b;
       b=a-b;
       a=a-b;
}


  • swap1: Si a l’inici a=A i b=B, al sortir a=B i b=B, les dues variables prenen el valor del d’entrada de b
  • swap2: Intercanvia el valor dels dos paràmetres (que són d’entrada). Per tant, no es produeixen canvis on s’ha fet la crida.
  • swap3: Fa l’intercanvi dels valors de les dues variables. Si a l’inici a=A i b=B, al final a=B i b=A.
    Fent la traça obtenim:
        a = A+B
        b = (A+B)-B = A
        a = (A+B)-A = B
    Si es fa la crida amb la mateixa variable, llavors a=X, b=X (a i b són dos àlies de la mateixa variable, fan referència a la mateixa posició de memòria). Per tant obtindríem un 0 de valor final ja que:
        a = X + X = 2X (ara b també val 2X)
        b = 2X -2X = 0 (ara a també val 0)
        a = 0 –0 = 0
