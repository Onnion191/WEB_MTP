5.3 ESPECIFICACIÓ

Un algorisme és una successió d'instruccions que modifiquen l'entorn, una successió d'estats definits pels valors de les variables de l'entorn en un moment donat.
La pre i postcondició formalment es defineixen per predicats, expressions booleanes (amb quantificadors), que representen el conjunt de valors que fan que el predicat sigui cert. Quan un algorisme complex una especificiació {P}S{Q} (essent P la pre, S les sentències i Q la post), es guaranteix que si l'algorisme comença en algun dels estats de P, acabarà en algun dels estats de Q.

Invariants d’un programa i disseny iteratiu

L’especificació i els invariants permeten fer una anàlisi acurada dels algorismes que requereixen disseny iteratiu. 

unsigned potencia(unsigned base, unsigned exponent) {
//Pre: base>0
//Post: retorna base^exponent

    unsigned resultat=1;
    unsigned comptador=0;
    // Invariant: resultat=base^comptador i comptador<=exponent
    while (comptador<exponent) {
        // Invariant: resultat=base^comptador i comptador<exponent
        resultat=resultat*base;
        comptador++;
        // Invariant: resultat=base^comptador i comptador<=exponent
    }
    // Invariant: resultat=base^exponent
    return resultat;
}
