INSTRUCCIONS BÀSIQUES

Quines són?

Les instruccions bàsiques són aquelles que es consideren necessàries per fer el programa més senzill. Aquestes instruccions són: Assignar valors, Escriure i Llegir:
  • Assignar valors: actualitzar el valor d'una variable donant-li un nou valor.
  • Escriure: mostrar a l'usuari, normalment via pantalla, informació sobre el programa i/o resultats d'execució.
  • Llegir: llegir les dades que li proporciona l'usuari, normalment via teclat.
Per executar instruccions en C++ cal seguir unes pautes per tal d'evitar errors de compilació:
  • Totes les instruccions acaben amb un ; al final. (exemple: llegir(); escriure();). Si no es posen, no podrem compil·lar el programa.
  • Les instruccions es llegeixen de dalt a baix i de forma ordenada. Això obliga a declarar variables abans de poder-les utilitzar.

Assignació

La assignació és el procés de canviar el valor a una variable, sobreescrivint el valor anterior. En C++ utilitzem el = per assignar el valor de la dreta, a la variable de l'esquerra. Com sabem aquestes variables permeten fer operacions entre elles, en el cas de les numèriques tenim les operacions bàsiques de les matemàtiques. El C++ ens permet utilitzar aquestes operacions de diferents formes, les explícites i les compactes:
  • Explícites: l'usuari escriu ell mateix com vol que sgigui els operands, i assigna el resultat fina la una variable. Per exemple: 
    
                int a = 5; //Inicialització de a = 5; 
                a = a * 2;// Operació explicita a * 2.
  • Compactes: les compactes només necessitem un operand ja que el segon operand es sobreenten que és sobre la variable que estem fent la assignació. 
    
                int a = 5; //Inicialització de a = 5; 
                a*=2; // Operacion compacte, multipliquem a*2 i assignem el resultat a " a ".

Escriptura

El procés d'escriptura consisteix en mostrar informació al usuari, el mètode més normal és mostrar aquesta informació per la pantalla. En c++ utilitzarem la llibreria <iostream> i necessitem utilitzar el nom d'espais std. 

        #include <iostream>
        
        using namespace std;
La instrucció que utilitzarem és cout que permet mostrar per pantalla la informació que desitja l'usuari. Per utilitzar es fa de la següent manera: instruccio<<informacio a mostrar<<informacio a mostrar;

Com es pot veure hi ha dues informacions a mostrar, això és degut a que cada element que es vulgui mostrar ha d'estar separat per l'operador <<. Per tant, si volem mostrar dues variables utilitzarem la següent instrucció; 

            cout<<meva_variable<<meva_variable2<<endl;
endl s'utilitza per fer un salt de linia, d'aquesta manera la següent informació que es mostri sortirar en una nova linia.

Formateig de la sortida

Hi ha la possibilitat de triar com volem que surti la sortida, per exemple, escollir quants decimals volem que es mostrin, si volem que la linia ocupi una mida maxima, etc. Per poder utilitzar aquestes opcions, cal que afegim una nova llibreria <iomanip>. Algunes de les coses que es poden fer:
  • Escollir l'amplada: permet escollir quants caràcters es mostraràn per cada sortida. Entenem que cada element entre << és una sortida. 
    
                cout<<setw(8)<<2.3; // Mostrara:     2.3 (5 espais i 2.3 = 8);
    • Escollir quin caràcter omplir els forats en blanc: com hem vist abans, el setw ens genera espais en blanc. Podem escollir que en comptes d'espais en blanc, usi un altre caràcter 
      
                            cout<<setfill('.')<<set(8)<<2.3; //Mostrara:.....2.3 (5 punts i 2.3 = 8)
  • Escollir la precisió: podem triar quants decimals volem mostrar per pantalla. 
    
                        cout.setf(ios::fixed); 
                        cout.precision(2);//A partir d'ara tots els reals que es mostrin tindran 2 decimals.

Lectura

El procés de lectura (o entrada) consisteix en llegir dades externes al programa i guardar-les en una variable. És una de les instruccions més importants ja que permet la interacció entre programa - usuari que és gairebé obligada en molts casos. En el nostre cas actual ens centrarem en llegir dades des del teclat, malgrat que també les podem llegir des de altres fonts com podria ser: un fitxer, internet, etc.

En C++ usarem la instrucció cin que requereix incluir la llibreria <iomanip>. De la mateixa forma que en el cas de la sortida on utilitzavem la instruccio cout seguida de l'operador <<. En el nostre cas usarem cin seguit de l'operador >> que ens separarà les diferents entrades que volguem llegir.
Exemple: 

         #include <iostream>
            ...
            int var_entera;
            cin>>var_entera;//Llegim un enter per teclat
            ...
Podem veure que en funció de la variable on volguem guardar l'entrada, el programa llegirà un enter, un real, un caràcter, etc. Per això és important saber quines dades esperem que entrin i informar a l'usuari. Si intentem guardar un tipus de dada a una variable de diferent tipus, podem tenir resultats indesitjats.

Sovint abans de demanar una entrada de teclat s'informa a l'usuari quina dada ha d'introduïr: 

            #include <iostream>
            ...
            cout<<"Introdueix la teva edat i el teu nom"<<endl;
            
            unsigned int edat; 
            string nom;
            cin>>edat>>nom;//Llegeix un natural i el guarda a edat --> Llegeix una cadena de caracters i el guarda a nom.
            ...
