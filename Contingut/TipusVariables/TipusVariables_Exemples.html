Problema 3



Per facilitar la tasca de càlcul de preus amb descompte i la gestió d'una botiga, es vol fer un programa que calculi el preu final d'una compra a la que cal aplicar un descompte, i també el canvi que cal tornar al client. Es vol fer sabent el preu de compra, el % de descompte i la quantitat lliurada pel client. El programa ens ha de calcular el preu final de la compra i el canvi que cal tornar. 

        #include<iostream>
        #include<iomanip>
        
        //Entrada: llegeix (preu inicial i diners) dos nombres reals i descompte un nombre natural
        //Sortida: mostra el preu final un cop aplicat el descompte a preu_inicial i el canvi que cal tornar d'acord amb el preu final i els diners lliurats        
        int main(){
            cout<<"Preu: "<<endl;
            double preu_inicial;
            cin>>preu_inicial; 
            
            cout<<"Descompte: ";
            unsigned descompte; 
            cin>>descompte;
            
            double a_pagar = preu_inicial*(1-descompte/100);
            a_pagar = int(a_pagar*100+0.5)/100;
            cout.setf(ios::fixed);
            cout.precision(2);
            cout<<"Total :"<<a_pagar<<endl;
            
            cout<<"Quantitat Lliurada: ";
            double diners; 
            cin>>diners;
            
            double canvi = diners - a_pagar; 
            cout<<"Canvi: "<<canvi<<endl;
            
            return 0;
        }
