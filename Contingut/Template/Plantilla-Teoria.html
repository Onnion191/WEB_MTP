Estructura de la web

La web està estructurades per directoris que tenen la següent composició:

SUBTITULS

Per introduir teoria utilitzem el tag teoria <teoria> ... contingut .... </teoria>
Els blocs de teoria:
  • S'obren sempre que s'escriu contingut en una pàgina de teoria.
  • Es tanquen després d'acabar aquell bloc. Sovint un bloc correspon a l'explicació de cada subtítol.
  • Si es vol introduir linies d'alerta, per remarcar el que calgui: <alerta> contingut alerta </alerta>


Podem introduir talls de codi dins de la teoria, per fer-ho necessitem fer una cadena de tags:
  • Pseudocodi: si volem fer un tros de pseudocodi els tags són els següents: <pseudocodi><pre><code class="nohighlight"> .... codi ... </code ></pre></pseudocodi> 
    
                    Aixo és un exemple de pseudocodi
  • C++: si volem fer un tros de c++ únicament hem d'esborrar el pseudocodi, i canviar la classe del code al llenguatge que volem utilitzar, el nostre cas c++ <pre><code class="c++"> .... codi ... </code ></pre> 
    
                  "Aixo es un exemple de c++"
    • Els blocs de codi només s'actualitzen si es refresca la pàgina

Codi amagat darrera un botó

Si volem fer que un codi només es vegi si es clica un botó, hem de fer exactament el mateix que quan afegim un tros de codi, però amb un tag extra i la creació del boto html.
  • Creació del boto: el botó es crearà de la següent manera: <button onclick="myFuntion(IDcodi)"> nom visible del boto </button>
    El idCodi serà l'idintificador del nostre tros de codi, ha de ser ÚNIC en tota la pàgina, altrament tindrem un error de javascript.
  • Creació del codi: per fer el codi, utilitzarem els següents tags: <solucio id="idCodi"><pre><code class="c++"> .... codi ... </code ></pre></solucio>
    Un exemple seria el següent: 
    
              "Exemple codi C++"
            
    
             Exemple Pseudocodi

Butons a Github

Si en comptes de voler un botó que mostri el codi, volem anar al github, usem el següent: <button> <a target="_blank" href= "link a pàgina externa"> Veure Solució </a> </button>

SEPARADOR TOTAL

Tenim la opció de trencar la pàgina web afegint un separador total: <h1> ... contingut .... </h1>
