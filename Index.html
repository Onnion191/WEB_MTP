Què és una variable?

En C++ podem entendre una variable com un contenidor finit on podem emmagatzamar diferents dades. Aquestes dades necessiten tenir un format per tal d'entendre què estem tractant, no és el mateix emmagatzamar lletres, que emmagatzamar nombres decimals.

Aquests contenidors tenen una mida finita, és a dir, no poden guardar qualsevol cosa i de qualsevol mida. Aquesta limitació s'ha de tenir en compte, per tal d'adecuar al màxim les nostre variables al que realment volem utilitzar.

D'aquí és on neix la necessitat definir quins tipus de variables podem emmagatzamar.

Tipus de Variables

Numèriques:

Ens permet guardar valors numèrics, però hi ha diferents maneres de representar els nombres:
  • Naturals, només permeten valors positius i sense part decimal.
  • Enters, permeten tant positius i negatius però sense part decimal
  • Reals, permeten decimals positius i negatius.
Com podem observar els reals són un comodí quan no saps quin tipus de dada estàs tractant, però cal tenir en compte que com més flexibilitat tenim, més memòria ocuparà al sistema, i això podria ser un inconvenient.

Lògiques:

Aquest tipus de variables està dissenyat per guardar valors com cert o fals, i permet fer diferents operacions lògiques amb aquestes operacions. En aquest cas, si entenem la variable booleà com un contenidor, aleshores simplement ocupariem un byte (8 bits). La raó per la qual ocupa 8 bits en comptes de 1, és perquè en c++ és necessàri que poguem referenciar el "contenidor" i el tamany mínim per això és un byte.

Implementació:



        int main(){
            bool a; //Declarem un boolea. El contingut és incert perquè no esta inicialitzat
            a = false; //Assignem false a " a ".
            
            bool b = true;//Declarem i inicialitzem b a true. El contingut es True. 
            
            //Recordem que 0 = false i 1 = true
            
            bool c; //Declarem un boolea c. El contingut és incert.
            
            c = !a; // Assignem no " a "  a c. C = true;
            cout<<c<<endl;
            c = a || b; //Assignem " a " o " b " a " c ". Si un dels dos (a,b) és cert, aleshores C és cert.
            cout<<c<<endl;
            c = a && b;//Assignem " a " i " b " a " c". Si a i b són certs, aleshores C és cert.
            cout<<c<<endl;
            
            return 0;
        }
